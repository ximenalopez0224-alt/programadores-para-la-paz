Al ejecutar el comando node server.js, Node.js inicia la aplicación y pone el servidor en funcionamiento.

En la terminal aparece el mensaje:
"Servidor ejecutándose en puerto 3000"

Cuando se visita la ruta http://localhost:3000 se muestra el mensaje:
"Servidor activo y funcionando"

Cuando se visita la ruta http://localhost:3000/estado se obtiene la siguiente respuesta:

{
"estado": "Servidor funcionando",
"servicio": "API comunitaria"
}

Esto indica que el servidor está respondiendo correctamente a las solicitudes.

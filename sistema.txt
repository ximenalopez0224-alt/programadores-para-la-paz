Comando: uname -a

Este comando muestra información detallada del sistema operativo, incluyendo el nombre del sistema, versión del kernel, arquitectura y otros datos técnicos.

Comando: whoami

Este comando muestra el nombre del usuario que actualmente está utilizando la sesión del sistema.

Importancia:

Conocer la información del sistema es útil para administrar servidores porque permite identificar el entorno de trabajo, verificar versiones del sistema operativo, realizar tareas de mantenimiento y solucionar problemas de forma más eficiente.

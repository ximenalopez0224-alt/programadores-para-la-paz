El login es el proceso mediante el cual un usuario se identifica en una aplicación usando un nombre de usuario o correo electrónico y una contraseña. Esto permite verificar su identidad antes de acceder a determinadas funciones. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar acciones importantes, como modificar o eliminar datos. Gracias a la autenticación, solo los usuarios autorizados pueden acceder a estas funciones. Además, nunca deben exponerse públicamente contraseñas, documentos de identidad, direcciones, teléfonos, tokens, claves de API ni otra información sensible de la comunidad.
